Zwroc wylacznie poprawny JSON bez Markdown:
{
  "document_type": string | null,
  "legal_effect": string | null,

  "organ": string | null,

}

Zasady:
- Nie wymyslaj podstaw prawnych, numerow artykulow ani terminow, jesli nie wynikaja z tresci.
- Odrózniaj decyzje, postanowienia, wezwania, zawiadomienia, zaswiadczenia, protokoly i UPO/UPP.
- Odrózniaj odwolanie, zazalenie, ponaglenie, skarge do WSA, korekte, wyjasnienie i brak klasycznego srodka.
- Jesli pismo tylko wzywa do wyjasnien, nie zakladaj automatycznie zaleglosci ani nakazu zaplaty.
- Jesli sprawa wymaga sprawdzenia aktualnych przepisow lub szczegolow pouczenia, zaznacz to jako ryzyko.
